슬라이더(slider : range)

개요(Overview)

슬라이더는 양수/음수, 소숫점을 모두 허용한 숫자로 범위를 지정하여 해당 범위 내로 숫자를 선택할 수 있게 해주는 요소로 HTML5의 type="range"가 원형이다. Android에서는 시크 바(SeekBar)라고 불린다.

비슷한 계열의 요소로는 다운로드, 페이지 로드와 같이 무언가를 기다릴 때 진행상황 업데이트를 표시하고 알리는 진행률 막대(ProgressBar), 그래프와 같은 정적인 내용을 표시하는 미터(meter) 요소가 있다.

주요 속성

속성 설명
aria-valuemin="int_string" input에서의 min과 동일한 역할을 구현하기 위해 사용합니다(필수).
aria-valuemax="int_string" input에서의 max와 동일한 역할을 구현하기 위해 사용합니다(필수).
aria-valuenow="int_string" input에서의 value와 동일한 역할을 구현하기 위해 사용합니다(필수).
aria-valuetext="string"

현재 값을 어떤 단위로 읽은 것인지 계산한 다음, 이 속성으로 전달하면 값을 조절할 때 현재 재생 시간이나 퍼센트 등으로 현재 슬라이더 값을 사용자에게 전달할 수 있습니다.

예):<custom-slider aria-valuetext="0점" />

입력장치로 조작 및 접근

슬라이더 조절

슬라이더 요소는 좌/우 화살표 키(code: 'ArrowLeft' / 'ArrowRight')로 슬라이더 수치를 조절하고, 조절된 값을 스크린리더를 통해 들을 수 있어야 한다.

HTML5의 range 입력서식에서는 PageUp과 PageDown으로도 슬라이더 값을 화살표 키를 누른 것과 동일한 값만큼을 조절 가능하며 구현하면 좋으나, 구현하지 않아도 큰 문제가 되진 않는다.

기본 입력서식에 따라 PageDown/PageUp 슬라이더 조절 구현 시, 동일한 수치로 조절하는 것도 좋으나 화살표 키로 조절하는 것과 슬라이더 조절 단위에 차별점을 두는 것또한 좋다.

슬라이더를 시작부분/끝 부분으로 조절(선택) HTML5의 range 입력서식에서는 Home 키로 슬라이더의 시작부분으로 핸들을 보내거나 슬라이더는 End 키로 슬라이더의 끝부분으로 핸들을 조절할 수 있다. 이를 동일하게 구현하는 것이 좋으나, 슬라이더의 용도에 따라 생략될 수 있다.
슬라이더 값 읽기

스크린리더 사용자는 Tab 키로 슬라이더에 초점을 보냈을 때와 오른쪽 또는 왼쪽 화살표 키로 슬라이더를 조절했을 때 값을 음성으로 들을 수 있어야 한다.

단, 재생시간 조절 슬라이더와 같이 값이 자동으로 업데이트되는 슬라이더는 Tab 키로 슬라이더에 초점을 보냈을 때 슬라이더의 값을 개속 읽어서는 안 되며, 초점을 받았을 때, 단 한 번만 알려야 한다. (Audio 객체의 timeupdate 이벤트 도중 aria-valuetext를 업데이트하지 말 것)

모바일 하드웨어키 조작 지원(선택)

모바일 스크린리더 사용자는 특정 제스처나 모바일의 물리적인 버튼으로 슬라이더 값을 조절할 수 있고, 개발자는 이를 지원할 수 있습니다.

현재로서는 모바일의 볼륨버튼이나 제스처를 인식하는 API가 웹에 존재하지 않아 이를 순수 커스텀으로 지원함에 어려움이 있습니다. 시각적으로 숨김처리한 input[type="range"]를 커스텀 슬라이더와 동기화시켜 모바일 제스처나 물리적 버튼 조작을 지원할 수 있습니다.

숨김처리한 range로 모바일 디바이스의 조작을 지원하려면 userAgent이 모바일인지 체크하여 PC로 접속중이면 display:none;으로 숨겨야 합니다.

참고자료: CSS로 range 숨기기: CSS 속성
  • appearance:none - 브라우저에서 제공하는 스타일을 제거합니다. 브라우저에 따라 접두사 -moz-, -webkit-등이 붙을 수 있습니다.
  • background-color:tranparent - 트랙 바의 색상을 투명하게 숨깁니다.
CSS 의사 클래스(pesudo-class)
  • input[type=range]::-moz-range-thumb - Firefox에서 슬라이더 핸들 스타일을 정의하는 의사 클래스입니다. -moz-appearance:none 을 제공하면 핸들을 숨길 수 있습니다.
  • input[type=range]::-webkit-slider-thumb - Chrome/Safari에서 슬라이더 핸들 스타일을 정의하는 의사 클래스입니다. -webkit-appearance:none 을 제공하면 핸들을 숨길 수 있습니다.
