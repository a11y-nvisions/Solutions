아코디언

개요(Overview)

아코디언 요소는 영역을 display나 visibility 등으로 표시하거나 숨기는 기능을 수행하는 요소를 얘기하며, 버튼, 메뉴, 트리뷰 등에 사용하는 개념으로, 접힘과 폄으로 연주하는 악기인 아코디언과 흡사하여 붙혀진 이름이며, 확장 및 축소 상태가 적용되는 요소를 가르킨다.

aria-expanded (필수)

스크린리더에 확장 및 축소 상태정보를 전달한다.
문법

<element aria-expanded="boolean"></element>
사용 태그(역할)
  • application
  • button
  • combobox
  • menuitem
  • treeitem
  • checkbox
속성값
true

영역 또는 하위 메뉴를 조작했을 때 확장됨을 음성으로 전달한다.
false

영역 또는 하위 메뉴를 조작했을 때 축소됨을 음성으로 전달한다.
undefined(기본값)

확장/축소 정보를 제공하지 않는다. 이 값은 사용되지 않는다. 해당 기능을 수행하려면 ElementObject.removeAttribute()메서드를 사용하여 상태정보를 제거해야 한다.
원칙
구조 영역 확장/축소 요소는 반드시 영역 시작 전에 위치해야 한다.(선형 콘텐츠)
키보드 이벤트 제공 트리뷰나 콤보박스, 메뉴와 같이 누르기 동작 외의 방향키와 같은 별도의 조작방법이 있는 요소의 경우, 동등하게 구현해주어야 한다.

풀다운, 트리뷰 : 오른쪽 / 오른쪽 방향키로 확장이 가능하고, 왼쪽 방향키나 esc키로 축소가 가능해야 함

드롭다운과 콤보박스 : 키보드를 통한 클릭 외에 아래 방향키로 확장이 가능하고, esc키로 축소가 가능해야 함,

축소상태의 콤보박스를 아래 방향키를 통해 아이템 선택이 가능한 경우 : Alt + 아래 방향키로 확장이 가능해야 함.
