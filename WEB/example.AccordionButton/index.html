아코디언

개요(Overview)

아코디언 요소는 영역을 display나 visibility 등으로 표시하거나 숨기는 기능을 수행하는 요소를 얘기하며, 버튼, 메뉴, 트리뷰 등에 사용하는 개념으로, 접힘과 폄으로 연주하는 악기인 아코디언과 흡사하여 붙여진 이름이며, 확장 및 축소 상태가 적용되는 요소를 가리킨다.

구현 방법 핵심 및 주요 개념

aria-expanded
속성값 속성값 설명
true

영역 또는 하위 메뉴를 조작했을 때 확장됨을 음성으로 전달한다.

false

영역 또는 하위 메뉴를 조작했을 때 축소됨을 음성으로 전달한다.

undefined (기본값)

확장/축소 정보를 제공하지 않는다. 이 값은 사용되지 않는다. 해당 기능을 수행하려면 ElementObject.removeAttribute()메서드를 사용하여 상태정보를 제거해야 한다.

사용 가능한 역할과 태그
역할(태그) 설명
application 확장/축소가 가능한 웹 애플리케이션 영역에 대한 상태정보를 제공할 수 있다.
button 특정 영역을 접거나 펼칠 수 있는 버튼에 상태정보를 제공할 수 있다.
combobox 콤보 상자의 option 목록 상자의 접힘과 펼침 상태정보를 제공할 수 있다.
menuitem 메뉴 항목의 하위 메뉴 목록에 대한 펼침 및 접힘 상태정보를 제공할 수 있다.
treeitem 트리뷰 항목의 하위 트리뷰 목록에 대한 펼침 및 접힘 상태정보를 제공할 수 있다.
checkbox 선택, 선택 안 함 상태에 따라 확장과 축소가 이루어지는 체크 상자 요소에 펼침 및 접힘 상태정보를 제공할 수 있다.
원칙
구조 영역 확장/축소 요소는 반드시 영역 시작 전에 위치해야 한다. (선형 콘텐츠)
키보드 이벤트 제공 트리뷰나 콤보 상자, 메뉴와 같이 누르기 동작 외의 화살표 키와 같은 별도의 조작 방법이 있는 요소의 경우, 동등하게 구현해주어야 한다.

풀다운, 트리뷰 : 오른쪽 / 오른쪽 화살표 키로 확장이 가능하고, 왼쪽 화살표 키나 ESC 키로 축소가 가능해야 함

드롭다운과 콤보 상자 : 키보드를 통한 클릭 외에 아래 화살표 키로 확장이 가능하고, ESC 키로 축소가 가능해야 함,

축소상태의 콤보 상자를 아래 화살표 키를 통해 아이템 선택이 가능한 경우 : Alt + 아래 화살표 키로 확장이 가능해야 함.
